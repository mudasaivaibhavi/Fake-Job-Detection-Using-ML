# Fake-Job-Detection-Using-ML
An ML-based project that detects fake job postings using NLP and classification algorithms. Built with Python, Scikit-learn, TF-IDF Vectorization, and Naive Bayes.

Features

  Fake vs Real job prediction,
  Text preprocessing & NLP,
  TF-IDF feature extraction,
  Naive Bayes classification,
  Model evaluation and accuracy analysis.
  
Tech Stack

  Python,
  Scikit-learn,
  Pandas,
  NumPy.
  
Built to identify suspicious job listings because the internet keeps producing “earn ₹2 lakh/week from home” scams like it’s an Olympic sport.
